FATAL 错误合集

    终于终于装好了，恭喜FATAL加入web最难忘单词大家庭！

     搭了这么久hexo，关于怎么搭建，我就不一一说明了，小洪童鞋和小李童鞋写的搭建指南很全面，大家可以去参考学习。这篇博客的主要目的就是把我遇到的所有错误整一个合集,希望能帮你们快速的找到搭建时出现的错误解决方法㊥
     来来回回装了好几遍，每次都会有不同的错误，所以顺序会有变化，大家就对号入座找错误吧

1. fatal: The remote end hung up unexpectedly 原因是推送的文件太大。
解决方法：☞网页链接1
 2.npm ERR! code ELIFECYCLE npm ERR! errno 1 npm ERR! ejs@2.7.4 postinstall: `node ./postinstall.js` npm ERR! Exit status 1
解决方法：将 npm install 改为 npm install --ignore-scripts
3.npm ERR! Cannot read property 'match' of undefined
解决方法：我把项目文件夹下面的package-lock.json文件删掉就可以了，参考☞网页链接2
 4.FATAL:Node Sass does not yet support your current environment: Windows 64-bit
解决方法：网页链接3
 5.git clone时RPC failed; curl 56 OpenSSL SSL_read: SSL_ERROR_SYSCALL, errno 10054
解决方法：☞网页链接4
6.FATAL:can not read a block mapping entry; a multiline key may not be an implicit key at line
解决方法：☞网页链接5，一定要注意空格哦~
 7.npm ERR! code ENOTFOUND npm ERR! errno ENOTFOUND npm ERR! network request to https://registry.npmjs.org/这个是我第一次搭建时出现的后面就没有出现过了
解决方法：☞网页链接6

8.npm ERR! Unexpected end of JSON input while parsing near这应该也是不太常见的错误
解决方法：☞网页链接7

以上就是我出现过的错误，搭一个博客真是一把辛酸泪,希望大家都能搭好自己的hexo,biubiubiu~
