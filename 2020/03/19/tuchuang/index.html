一起来搭个图床

用GitHub+picgo搭一个图床

GitHub 仓库设置

1.1 新建仓库

点击 git 主页右上角的 + 创建 New repository；

1.2创建 token 并复制保存


在页面最下找到 Developer settings，点击进入；

填 description（也是随心填），勾选复选框 repo ，接着到页面底部 Generate token 就完成了；

复制生成一串字符 token，这个 token 只出现一次，所以要保存一下


PicGo 客户端配置

2.1 下载&安装

百度云链接：百度云密码：PicG

2.2配置


域名修改为:https://raw.githubusercontent.com/[username]/[仓库名]/master

上传的快捷键设置。默认的是 Mac 按键，推荐改成 Ctrl+c

然后图床就搞好啦
